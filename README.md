## Commands:

### JobConfig

Configures the job timer.

```
!JobConfig Get
!jobconfig help (or just !jobconfig)
!!JobConfig Set 5 Seconds
!jobconfig set 10 minutes
!JoBcOnFiG sEt 1 HoUr
!jobconfig remove
```

### SetRandom

Want the bot to post a random gif if there's no new trending one?

```
!SetRandom On
!setrandom on cats
!SeTrAnDoM oFf
```
